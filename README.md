## 📚 What I Learned

### Day 01: Digital Foundation
- เข้าใจภาพรวมของอาชีพ Software Engineer และสายงานด้านเทคโนโลยี
- เข้าใจองค์ประกอบของคอมพิวเตอร์และการทำงานของระบบปฏิบัติการ
- รู้จักแนวคิด Input, Process, Output (IPO)


### Day 02: Logic & Flowgorithm Lab 1
- ฝึกคิดวิเคราะห์ปัญหาอย่างเป็นขั้นตอน (Logical Thinking)
- เรียนรู้การออกแบบ Algorithm และ Pseudocode
- ใช้โปรแกรม Flowgorithm สร้าง Flowchart

### Day 03: Flowgorithm Lab 2–3
- เรียนรู้โครงสร้างการตัดสินใจ (If, Else If, Else)
- เข้าใจการแปลง Flowchart ไปเป็น Source Code เบื้องต้น
- เรียนรู้การใช้ Loop และ Counter

### Day 04: AI, Web Technology and Network
- เรียนรู้พื้นฐาน AI และการประยุกต์ใช้ AI ในงาน Software Engineering
- เข้าใจโครงสร้างของเว็บไซต์ เช่น Frontend, Backend, Database และ API
- เรียนรู้พื้นฐานเครือข่ายคอมพิวเตอร์
